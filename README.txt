1. Open the solution in Visual Studio, preferably 2017.
2. Press Ctrl + F5 to run the solution.
3. It is assumed to have a 16 by 16 grid of cells, where each cell is randomly set as Live (L) or Dead (D), initiallly, using the Random Number Generator.
4. Rules of the Conways Game are applied at every tick, which is 2 seconds, indefnitely.
5. The game can be stopped anytime using the 'ESC' (escape) button.
